Informations sur les versions de developpement

Les versions de développement sont des versions non terminées, potentiellement instables. Ce sont les dernières compilations faites pendant le developpement du jeu, et n'ont donc pas toutes les fonctionalités prévues pour la version indiquée.

Ces versions sont utiles à ceux voulant découvrir ou tester des nouvelles fonctionalités en avance.

Le codage de ces versions se fait ainsi : V dev JJMML
V = La version vers laquelle se dirige la version de developpement
dev = Le mot clé indiquant le numéro de la version de developpement
JJ = Le jour de la publication de la version de developpement
MM = Le mois de la publication de la version de developpement
L = La lettre identifiant la version de developpement. Si deux versions de developpement sont publiées le meme jour, par exemple le 26 avril, leur numéro sera respectivement 2604a et 2604b
