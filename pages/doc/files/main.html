Macros

PRINT_TICKS

PRINT_TICKS permet d'afficher le nombre de ticks écoulés depuis l'initialisation de SDL. Est surtout utilisé dans le log.

UNS

UNS est une manière plus rapide d'écrire using namespace std;

Variables

std::ofstream rlog

rlog est le ofstream du log du programme.

std::ofstream rerrLog

rerrLog est le ofstream du log d'erreur du programme. La fonction gererErreur() utilise cet objet.

Fonctions

void gererErreur(std::string erreur, bool fatal)

gererErreur permet de gerer une erreur et de l'inscrire dans le log d'erreur.
Paramètres :

  • std::string erreur : La chaine de caractère décrivant l'erreur
  • bool fatal : Si ce paramètre est true, cela éteindra le programme.

int quit(int sortie)

quit permet de quitter le programme en fermant les flux de logs et en quittant SDL. Le paramètre sortie est renvoyé en sortie du programme.

int main(int, char*[])

Fonction démarrant le programme.

Namespaces

namespace Main

Namespace contenant des éléments pour faire démarrer le jeu. Tous ses éléments sont dans le .cpp, puisqu'ils ne sont pas sensés servir autre part dans le programme.
