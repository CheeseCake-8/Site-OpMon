Macros

#define OP_NUMBER

Constante contenant le nombre d'OpMon au total.

#define ATK_LIST(OpId)

Macro aidant a la déclaration des attaques par niveau. Renvoie le nom d'un tableau contenant les attaques selon le numéro donné en paramètre.

#define ATK_ADD_LIST(OpID, atkLenght)

Ajoute les attaques du tableau contenant les attaques de l'OpMon dans le vecteur pour l'envoyer dans l'objet Espece.

Namespace

namespace Initializer

Contient les methodes nécessaires pour Initialiser et charger en mémoire les composants du programme. Contient les variables qui stockent ces derniers.
