Macros

ATK_LIST(OpId)

Macro permettant de raccourcir l'ajout des attaques. OpId est l'identifiant de l'espece OpMon a qui rajouter l'attaque dans le tableau listeOp. Il faut ensuite faire suivre ce macro de ATK_LIST(ID) = {NumberedArray(attaque, niveau), NumberedArray(attaque, niveau), ect...}.

ATK_ADD_LIST(OpID, atkLength)

Macro permettant de raccourcir l'ajout des attaques initialisées grace a ATK_LIST. OpID est le numéro de l'espece OpMon dans le tableau listeOp et atkLength représente le nombre d'attaques par niveau entrées dans le tableau déclaré avec ATK_LIST

OP_NUMBER

Constante contenant le nombre total d'especes de OpMon, et qui permet de regler la taille des tableaux.

Namespace

namespace Initializer

Namespace contenant les fonctions et variables permettant de charger et d'initialiser les éléments du jeu.

Contenu :

  • Espece *listeOp[OP_NUMBER]

    Tableau de pointeurs contenant la liste des Especes OpMon du jeu.

  • std::vector<NumberedArray<Class<Attaque* > > > opOpLvl[OP_NUMBER]

    Contient les attaques par niveau des especes.

  • std::vector<int> evs[OP_NUMBER]

    Contient les Ev de tous les OpMon.

  • void init()

    Initialise tous les éléments du jeu.
