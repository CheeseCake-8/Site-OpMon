Namespace

namespace StringKeys

Ce namespace contient toutes les methodes affiliées a StringKeys. StringKeys permet de lire des chaines de caractère dans un fichier et de les associer a des clées, permettant une traduction de ces chaines de caractères sans besoin de recompilation et de recherche dans le code.

Contenu:

  • std::ifstream keysFile

    Flux d'entrée sur le fichier contenant les chaines de caractères.

  • std::vector<std::string> keys

    Vector contenant les clées associées aux chaines de caractères.

  • std::vector<std::string> strings

    Vector contenant les chaines de caractère, dans le même ordre que le vector des clées.

  • std::string get(std::string keys)

    Fonction permettant de récuperer une chaine de caractères a partir de la clée associée, envoyée en paramètre (key). Si la clée n'existe pas, renvoie une chaine vide.

  • void initialize()

    Lit et enregistre les clées et les chaines de caractères.

  • std::string split(std::string str, char splitter, int part)

    Fonction séparant une chaine de caractère par rapport a un caractère et renvoyant une partie.
    Paramètres:

    • std::string str : La chaine de caractère a separer.
    • char splitter : Le caractère spéparant la chaine.
    • int part: La partie de la chaine séparée a renvoyer. Si le nombre est au dessus du nombre de parties, renvoie une chaine vide.
