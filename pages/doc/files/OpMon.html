namespace CalcCourbes

Ce namespace contient des fonctions permettant de calculer sur les courbes d'experience des OpMon.

      Constructeur permettant de construire un OpMon. Les paramètres sont associés aux variables du même nom.

    • bool captured(I_Opball &opball)

      Retourne true si l'OpMon est capturé en utilisant l'opball donné en paramètre.

    • void setStat(std::string &stat, int newStat)

      Permet de changer une statistique par le paramètre newStat, selon la chaine de caractère entrée :

      • ATK : Change l'attaque
      • DEF : Change la défense
      • ATKSPE : Change l'attaque spéciale
      • DEFSPE : Change la défense spéciale
      • VIT : Change la vitesse
      • PV : Change les PV max

    • void levelUp()

      Methode appelée lors d'une montée de niveau

    • bool isHoldingItem() const

      Permet de vérifier si l'OpMon tient un item

    • int win(OpMon &vaincu)

      Methode appelée lorsque l'OpMon gagne un combat. Le paramètre contient une référence vers l'OpMon vaincu.

    • void calcStats

      Recalcule les statistiques a partir de l'espece et du caractère

    • bool itemUsed(Item *used)

      Methode appelée lors de l'utilisation d'un item sur l'OpMon. Elle retourne true si l'item doit être supprimé de l'inventaire du joueur

    • Item* hold(Item *item)

      Permet de faire tenir un objet a l'OpMon. Renvoie l'item précédemment tenu, ou NULL si aucun item n'était présent. Pour retirer un item, il est possible de mettre NULL en paramètre.

    • void traded()

      Methode appelée lors d'un échange

      Permet de changer le status. Retourne false si le status n'a pas pu être changé

    • void heal(int PV)

      Soigne l'OpMon avec le nombre de PV donné en paramètre

    • bool operator==(OpMon &a)

      Cet opérateur ne compare pas deux OpMon mais verifie si l'OpMon comparé est falsif ou non. Si les deux sont falsifs, retourne true, si les deux ne sont pas falsifs, retourne true, dans le reste des cas, retourne false.

      Renvoie l'inverse de operator==
