Classe

class Espece

Définit une Espece d'OpMon.

Contenu :
  • PRIVATE:

    • std::string nom

      Contient le nom de l'Espece d'OpMon

    • int numeroOpdex

      Contient le numero qui identifie l'Espece dans le Opdex

    • Espece *evolution

      Contient un pointeur vers l'espece en laquelle cette espece evolue

    • int niveauEvolution

      Contient le niveau auquel cette Espece evolue (Si elle evolue par niveau, sinon cette variable vaut -1)

    • Evolution *evolType

      Contient le type d'évolution de cette Espece (voir evolutions.hpp)

    • int type1

      Contient le type premier de l'espece. Attend un élément de l'énumération "Type"
