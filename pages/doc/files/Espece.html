Classe

class Espece

Définit une Espece d'OpMon.

Contenu :
  • PRIVATE:

    • std::string nom

      Contient le nom de l'Espece d'OpMon

    • int numeroOpdex

      Contient le numero qui identifie l'Espece dans le Opdex

    • Espece *evolution

      Contient un pointeur vers l'espece en laquelle cette espece evolue

    • int niveauEvolution

      Contient le niveau auquel cette Espece evolue (Si elle evolue par niveau, sinon cette variable vaut -1)

    • Evolution *evolType

      Contient le type d'évolution de cette Espece (voir evolutions.hpp)

    • int type1

      Contient le type premier de l'espece. Attend un élément de l'énumération "Type"

    • int type2

      Contient le second type de l'espece. Attend un élément de l'énumération "Type"

    • NumberedArray atksByLevels[]

      Contient les attaques et les niveaux associées aux apprentissages de celles-ci par l'espece

    • std::string entreeOpdex

      Contient la description opdex de l'espece d'OpMon

    • float taille

      Contient la taille de l'OpMon

    • float poids

      Contient le poids de l'Opmon

    • CT ctCompatibles[]

      Contient les CT compatibles avec l'Espece

    • int baseAtk

      Contient la statistique de base d'attaque de l'espece

    • int baseAtk

      Contient la statistique de base de défense de l'espece

    • int baseAtk

      Contient la statistique de base d'attaque spéciale de l'espece

    • int baseAtk

      Contient la statistique de base de défense sépciale de l'espece

    • int baseAtk

      Contient les PV de base de l'espece

    • int baseVit

      Contient la statistique de base de vitesse de l'espece

    • int courbe

      Contient la courbe d'exp de l'espece. Attend un élément de l'énumération CourbeExp

    • int expMax

      Contient l'exp de l'espece d'OpMon au niveau 100

    • int *EVgiven

      Tableau contenant les EV donnés lors de la victoire sur cette espece

    • int expGiven

      Contient les exp de base donnés lors de la victoire du cette espece

    • int tauxDeCapture

      Contient le taux de capture de l'espece

    • int evSize

      Contient la taille du tableau d'ev (EVgiven)

  • PUBLIC

    • Espece(int atk, int def, int atkSpe, int defSpe, int vit, int pv, std::string name, int type1, int type2, int maniereEvolution, int niveauEvolution, Evolution *evolType, std::vector EVGiven, float taille, float poids, std::string entreeOpdex, int expGiven, int expMax, int tauxDeCapture, int numeroOpdex)

      Constructeur d'espece. Les noms des paramètres sont normalement assez explicites, mais quelques indications sur ceux-ci:

      • EVGiven contient la liste des ev donnés de l'espece qui seront mis dans EVgiven
      • niveauEvolution doit être a -1 si l'OpMon n'évolue pas par niveau (ou n'évolue pas tout court)

  • void checkEvol()

    Cette methode permet de récuperer l'espece d'évolution après l'initialisation de toutes les especes

  • void checkAtkLvls()

    Cette methode permet de récuperer les attaques par niveau pour les initialiser en différé de l'initialisation de l'espece
