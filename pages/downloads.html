Comment jouer ?

Sur Windows : Cliquez sur setup.exe et suivez les instructions, une fois l'installation terminée, lancez le jeu
Sur macOS et Linux : Telechargez Wine (version 1.6 ou supérieure) puis suivez les mêmes instructions que pour Windows
