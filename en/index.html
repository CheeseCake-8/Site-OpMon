Developed by a volunteer team

OpMon is developed by a team completely composed of volunteers! We ask nothing, but your happiness!

An open source game

OPMon is totally open source, and we even use open source software to create it: Linux, Code::blocks, Gimp, Git and many others!

Created by fans for fans

The game contains plenty of small Easters Eggs and winks to the universe of Nintendo and mediavisual while adding new features that restore a wind of freshness to the license.

Game totally free

The game was imagined in 2012 and made concrete in 2016 by a very small team of fans, the game is free, the source code is open to all, so you can come and help us and improve the game.
